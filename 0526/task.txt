TCP 소켓을 이용하면 서버/클라이언트 프로그램 작성
클라이언트를 실행하면
- 서버접속(s), 종료(x):
 s: 서버에 접속하고 접속/실패 메시지 표시
 x: 프로그램 종료
- 서버 접속에 성공한 후에는 로그인하도록 한다
 로그인: id pwd
- 서버에서는 로그인 정보를 수신하여 데이터베이스에서 이용자 인증 서버는 인증 결과를 클라이언트에게 전송한다
  클라이언트는 인증결과를 수신하고 화면에 로그인 성공/실패 표시
  로그인에 성공한 경우에만 다중 이용자 채팅에 참여할 수 있다

 채팅에 참여하는 이용자가 메시지를 서버로 바로 전송할 때는 이용자의 id가
 메시지 아페 붙어서 전송되도록 한다
